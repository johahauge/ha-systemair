# ha-systemair
Systemair modbus yaml-file for Home Assistant with modbus over TCP.

Applies to older models:
    VR400
    VR700
    VR700DK
    VR400DE
    VTC300
    VTC700
    VTR150K
    VTR200B
    VSR300
    VSR500
    VSR150
    VTR300
    VTR500
    VSR300DE
    VTC200
    VTC100

Reference the Systemair modbus reference PDF for more information and registers.
